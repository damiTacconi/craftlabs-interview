# Products Challenge

This repository contains the backend solution for the CraftLabs interview challenge.

It is organized as a modular .NET solution with:
- `Modules.Products`: core product domain contracts, DTOs, and services.
- `Modules.Products.Infrastructure`: infrastructure implementations (for example, in-memory repositories).
- `src/Products.WebAPI`: ASP.NET Core Web API host exposing product endpoints.
